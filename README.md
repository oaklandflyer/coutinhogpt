# Andrew Coutinho — 3D Interactive Portfolio

Immersive portfolio experience built with **React + Vite + React Three Fiber + Framer Motion**.

## Experience-first architecture

The implementation starts with a 3D interaction blueprint (scene chapters, camera path, scroll/mouse controls), then layers curated recruiter-facing content on top.

- Blueprint document: `docs/experience-model.md`
- 3D scene system: `src/scene/WorldScene.jsx`
- Overlay chapter UI: `src/components/OverlayUI.jsx`

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```
